01

슬롯 해부도

셀 하나에 들어가는 모든 것. 배지를 늘리지 마세요 — 슬롯은 조용해야 그리드가 읽힙니다.

24
1 슬롯 배경 — 누끼 이미지가 뜨지 않게 받쳐줍니다. 지금은 이게 없어서 흰 바탕에 그림이 떠 있어요. #E9E7E1 · radius 6 · border 1px #E1DED5
2 정사각 1:1 — 세로 0.6은 옷 전용 비율입니다. 정사각이라야 피규어·향수·LP도 어색하지 않아요. childAspectRatio: 1.0
3 즐겨찾기 = 우상단 붉은 삼각 — 아이콘의 빨간 칸과 대응. 라벨도 아이콘도 아닌 모서리를 잘라낸 형태라 이미지를 가리지 않습니다. 12×12 · #DE3B26
4 세탁 필요 = 좌하단 점 — 기존의 파란 “🧼 세탁 필요” 알약을 점 하나로. 파랑은 팔레트에 없는 색입니다. 6×6 dot · #DE3B26
5 착용 횟수 = 모노스페이스 — 인벤토리 톤을 만드는 건 색이 아니라 이 서체예요. mono · tabular-nums · 0이면 숨김
6 텍스트 라벨 없음 — 지금은 셀 밑에 “블랙 무지 / 상의·티셔츠” 2줄이 붙어 있습니다. 그리드는 눈으로 훑는 곳이에요. 이름은 상세에서 보면 됩니다. 지우면 슬롯이 커지고 훨씬 시원해져요.

04

구현 골자

기존 GestureDetector → Column 구조를 _SlotTile로 교체합니다.

// 1) GridView delegate — 세로 카드에서 정사각 격자로
SliverGridDelegateWithFixedCrossAxisCount(
  crossAxisCount: 3,
  crossAxisSpacing: 7,
  mainAxisSpacing: 7,
  childAspectRatio: 1.0,   // was 0.6
)

// 2) itemCount에 빈 슬롯 한 칸 추가
itemCount: docs.length + 1,
itemBuilder: (context, i) => i == docs.length
    ? const _EmptySlot()          // 점선 + "+" → UploadScreen
    : _SlotTile(doc: docs[i]),

// 3) 슬롯 타일 — 배경 있는 정사각 셀. 셀 하단 텍스트는 제거.
class _SlotTile extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return GestureDetector(
      onTap: ...,
      child: Container(
        decoration: BoxDecoration(
          color: AppColors.slot,                        // #E9E7E1
          borderRadius: BorderRadius.circular(6),
          border: Border.all(color: AppColors.slotLine), // #E1DED5
        ),
        clipBehavior: Clip.antiAlias,
        child: Stack(children: [
          Padding(
            padding: const EdgeInsets.all(10),           // 이미지가 숨 쉴 공간
            child: Hero(
              tag: docId,
              child: Image.network(imageUrl, fit: BoxFit.contain),
            ),
          ),
          // 즐겨찾기 — 우상단 붉은 삼각 (이미지를 안 가림)
          if (isFavorite)
            const Positioned(top: 0, right: 0,
              child: CustomPaint(size: Size(12, 12), painter: CornerNotch())),
          // 세탁 필요 — 좌하단 점 하나 (기존 파란 알약 대체)
          if (isWashRequired)
            Positioned(bottom: 5, left: 5, child: Container(
              width: 6, height: 6,
              decoration: const BoxDecoration(
                color: AppColors.accent, shape: BoxShape.circle))),
          // 착용 횟수 — 모노스페이스, 0이면 숨김
          if (wearCount > 0)
            Positioned(bottom: 3, right: 5, child: Text('$wearCount',
              style: AppText.mono.copyWith(fontSize: 9))),
        ]),
      ),
    );
  }
}
